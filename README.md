# slidecript_glsl

presentation text and glsl shader

自分用

## how to

* h1 : ヘッダ（既定では非表示）
* h2 : トップタイトル
* h3 : トップサブタイトル
* h4 : 見出し
* h5 : 非表示見出し（id が自動で付加される）
* h6 : 空き
* `@@@` : json
* `|||` : glsl

### json

```
{
    "title": {string},
    "author": {string},
    "description": {string}
}
```

### glsl

fragment shader only
