# slidecript_glsl

presentation text and glsl shader

自分用

## how to

* h1 : ヘッダ（既定では非表示）
* h2 : トップタイトル
* h3 : トップサブタイトル
* h4 : 見出し
* h4 : 見出し + 数字三桁以内（連番対象として処理される）
* `@@@` : json
* `|||` : glsl


### json

```
{
    "title": {string},
    "author": {string},
    "description": {string}
}
```

### glsl

fragment shader only

（glslsandbox 互換）
